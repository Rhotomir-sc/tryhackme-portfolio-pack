# TryHackMe Portfolio Pack 🛡️

**Goal:** Build HR-friendly evidence of hands-on cybersecurity skills via TryHackMe learning paths and mini-projects.

## What you’ll find here
- **Learning Paths:** structured progression (e.g., Pre-Security)
- **Room Evidence:** short writeups + reproducible outputs (screenshots, command outputs)
- **Mini-Projects:** small capstones to prove applied understanding

## Quick Links
- Progress Tracker → `PROGRESS.md`
- Roadmap → `ROADMAP.md`
- Evidence Index → `EVIDENCE_INDEX.md`

## Evidence Standard (My rule)
Each room includes:
- 3–7 key takeaways
- 2–5 proof items (screenshots/outputs)
- 1 small “applied step” (command, config, detection, or short analysis)
